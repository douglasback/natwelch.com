War! open help?

This is a javascript version of the card game War written by Nat Welch.

The game works by splitting the cards evenly among the two players using a Fisher-Yates Shuffle. Then The user drags his card into the middle of the game table.

Two cards are then displayed, one for the computer player and one for the user. The higher value card wins. Suit is irrelevent. Aces equal one and kings equal thirteen.

Refresh the page for a new game.

This was built using Mootools and Vim.
